# test-bimo: AI Exercise Video Pipeline

An automated pipeline for transcribing, segmenting, and categorizing exercise videos using WhisperX and LLMs (via OpenRouter).

## Overview

This project automates the workflow of turning long-form exercise videos into categorized, ranked, and named clips suitable for exercise libraries or social media.

### The Pipeline Flow
1. **Transcription:** Uses `WhisperX` to generate high-accuracy transcripts with timestamps.
2. **Segmentation:** Detects exercise boundaries based on transition keywords in the transcript.
3. **Naming:** Uses an LLM (Gemini/Claude via OpenRouter) to generate concise names for each exercise.
4. **Clipping:** Uses `ffmpeg` to extract individual exercises into high-quality clips.
5. **Metadata:** Logs all clips into a Google Sheet for easy management.
6. **Ranking:** Categorizes clips by body part and difficulty level using AI.

## Directory Structure

- `01_raw/`: Drop your input video files here (`.mp4`, `.mov`, etc.).
- `02_processing/`: Intermediate files (SRT, JSON, TSV) generated by WhisperX.
- `03_clips/`: The final extracted exercise clips.
- `04_done/`: Original source videos are moved here after processing.

## Setup

### Prerequisites
- **FFmpeg:** Required for video processing.
- **WhisperX:** Required for transcription.
- **Python 3.14+** (as specified in `pyproject.toml`).
- **Google Cloud Project:** With Sheets and Drive API enabled for metadata logging.

### Installation
This project uses `uv` for fast dependency management.
```bash
# Install dependencies
uv sync
```

### Configuration
1. **API Keys:** Create a `.env` file in the root directory:
   ```env
   OPENROUTER_API_KEY=your_api_key_here
   OPENROUTER_MODEL=google/gemini-flash-1.5
   ```
2. **Google Credentials:** Place your Google Service Account `credentials.json` in the root directory.
3. **Google Sheet:** Update `SHEET_ID` in `pipeline.py` and `ranker.py` to match your target spreadsheet.

## Usage

### 1. Run the Processing Pipeline
Processes all videos in `01_raw/`:
```bash
uv run pipeline.py
```

### 2. Run the Ranker
Categorizes unranked rows in the Google Sheet and renames files to include body parts and difficulty:
```bash
uv run ranker.py
```
